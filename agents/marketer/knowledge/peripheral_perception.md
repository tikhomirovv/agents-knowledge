# Периферийное восприятие

## Периферийное восприятие работает за 100 мс

### Суть
Периферийные визуальные раздражители активируют мозг уже в первые 100 мс через прямую связь с лимбической системой.

### Применение
- Создавать "эмоциональные аллеи" в интерфейсах
- Размещать позитивные элементы по краям экрана
- Использовать периферийное зрение для подсознательного воздействия
- Эмоциональные элементы на периферии влияют на центральный контент

### Практический совет
Периферийное восприятие работает быстрее сознания - используйте это.

## Позитивные периферийные раздражители повышают привлекательность

### Суть
Позитивные эмоциональные раздражители в зоне периферийного зрения делают основной продукт более привлекательным.

### Применение
- Размещать улыбки, детей, природу по краям экрана
- Создавать позитивную атмосферу вокруг основного контента
- Использовать смайлики и позитивные изображения на периферии
- Избегать негативных элементов в периферийной зоне

### Практический совет
Позитивная периферия = более привлекательный основной контент.

## Периферийное восприятие влияет на сознательные решения

### Суть
Подсознательные периферийные раздражители оказывают заметное влияние на сознательные когнитивные процессы и поведение.

### Применение
- Создавать правильное настроение через периферийные элементы
- Использовать подсознательное воздействие для влияния на решения
- Периферийные эмоции влияют на оценку основного контента
- Работать с настроением пользователя через периферийное восприятие

### Практический совет
Подсознательная периферия влияет на сознательные решения.

## Эмоциональная аллея в дизайне

### Суть
Создание "эмоциональной аллеи" - размещение позитивных элементов по краям для подсознательного воздействия.

### Применение
- Размещать позитивные элементы на расстоянии 10-15° от центра
- Создавать эмоциональную атмосферу вокруг основного контента
- Использовать периферийные элементы для управления настроением
- Планировать интерфейс с учетом периферийного восприятия

### Практический совет
Создавайте эмоциональные аллеи в интерфейсах для подсознательного воздействия.

## Фоновые эмоции влияют на привлекательность товара

### Суть
Позитивное эмоциональное окружение повышает привлекательность товара на 16% по сравнению с негативным контекстом.

### Применение
- Создавать позитивную атмосферу вокруг продуктов
- Использовать приятные цвета и изображения в фоне
- Избегать негативных элементов рядом с товарами
- Окружать продукты позитивными эмоциональными триггерами

### Практический совет
Позитивный фон = более привлекательный продукт.

## Фоновые эмоции влияют на ценовосприятие

### Суть
Позитивное эмоциональное окружение увеличивает готовность платить на 10% по сравнению с негативным контекстом.

### Применение
- Создавать позитивную атмосферу для повышения цены
- Использовать позитивные фоновые элементы в ценовых предложениях
- Избегать негативных эмоций при показе цен
- Окружать ценовые предложения позитивными образами

### Практический совет
Позитивный фон = готовность платить больше.

## Окситоцин и доверие к товару

### Суть
Позитивные визуальные раздражители вызывают выброс окситоцина, что создает положительное настроение и повышает доверие к товару.

### Применение
- Использовать изображения детей, влюбленных, природы
- Создавать атмосферу доверия через позитивные образы
- Активизировать выработку окситоцина у покупателей
- Связывать продукты с позитивными эмоциональными состояниями

### Практический совет
Окситоцин = доверие = готовность покупать.

## Негативные эмоции вызывают избегание

### Суть
Негативные фотографии вызывают эволюционную реакцию избегания, что снижает желание покупать и искажает ценовосприятие.

### Применение
- Избегать негативных образов в маркетинге
- Не использовать изображения войн, болезней, падающих акций
- Создавать атмосферу безопасности и комфорта
- Убирать все, что может вызвать негативные эмоции

### Практический совет
Негативные эмоции = избегание = потеря продаж.

## Самые лучшие покупатели - счастливые покупатели

### Суть
Счастливые покупатели готовы платить больше и воспринимают товары как более привлекательные.

### Применение
- Создавать счастье у покупателей
- Использовать позитивные эмоции для повышения продаж
- Делать покупки приятным опытом
- Фокусироваться на эмоциональном состоянии покупателей

### Практический совет
Счастье покупателей = больше продаж и выше цены.

## Эмоциональные ценники повышают привлекательность

### Суть
Эмоционально оформленные ценники повышают привлекательность товара на 21% (смайлик), 7% (влюбленная пара), 5% (удовольствие от потребления).

### Применение
- Использовать смайлики в оформлении ценников
- Добавлять эмоциональные образы к ценам
- Показывать удовольствие от потребления товара
- Создавать эмоциональную связь с продуктом

### Практический совет
Эмоции в ценниках = более привлекательный товар.

## Эмоциональные ценники повышают ценовосприятие

### Суть
Эмоциональное оформление ценников увеличивает привлекательность цены на 7% в среднем, смайлики - на 12%.

### Применение
- Использовать эмоциональные элементы в ценниках
- Смайлики работают лучше перечеркнутых цен
- Создавать позитивные эмоции вокруг цены
- Увеличивать доверие к ценообразованию через эмоции

### Практический совет
Эмоции в ценниках = готовность платить больше.

## Эффект смайлика - генетическая реакция

### Суть
Смайлики вызывают генетически обусловленную реакцию - выброс гормонов счастья при виде улыбающегося лица.

### Применение
- Использовать смайлики для максимального эмоционального воздействия
- Смайлики работают лучше других эмоциональных элементов
- Активизировать выброс гормонов счастья у покупателей
- Создавать генетически обусловленные позитивные реакции

### Практический совет
Смайлики = гормоны счастья = больше продаж.

## Эмоциональные фотографии направляют поток покупателей

### Суть
Эмоциональные фотографии увеличивают посещаемость отдела на 100% и товарооборот на 17%.

### Применение
- Размещать эмоциональные фотографии в стратегических местах
- Использовать большие форматы (125x187см) для максимального воздействия
- Направлять покупателей в нужные отделы через эмоции
- Создавать эмоциональные магниты для привлечения внимания

### Практический совет
Эмоциональные фотографии = направление потока покупателей.
