# Техники нейрокопирайтинга

## Описание

Этот файл содержит конкретные техники и приемы нейрокопирайтинга, основанные на понимании работы мозга и психологии потребителя.

## Основные разделы

### 1. Эмоциональные триггеры в тексте

**Слова-триггеры эмоций**
- Радость: "успех", "победа", "достижение", "результат"
- Страх: "риск", "потеря", "упущенная возможность", "проблема"
- Удивление: "секрет", "открытие", "неожиданно", "уникально"
- Жадность: "бесплатно", "скидка", "экономия", "выгода"

**Эмоциональные конструкции**
- "Представьте себе..." (включение воображения)
- "Что если..." (создание сценария)
- "Вы когда-нибудь..." (личный опыт)
- "Знаете ли вы..." (интрига)

### 2. Психологические принципы в тексте

**Принцип якорения**
- Сначала показывать высокую цену, затем низкую
- Использовать сравнения с дорогими аналогами
- Указывать стоимость часа работы эксперта

**Принцип контраста**
- "Было/стало" в описаниях результатов
- Сравнение "до и после"
- Противопоставление проблем и решений

**Принцип конкретности**
- Использовать точные цифры вместо общих фраз
- Описывать конкретные результаты
- Приводить детальные примеры

### 3. Структурные приемы

**Правило AIDA**
- Attention (внимание) - цепляющий заголовок
- Interest (интерес) - интригующее введение
- Desire (желание) - описание выгод
- Action (действие) - четкий призыв к действию

**Правило PAS**
- Problem (проблема) - описание боли клиента
- Agitate (усугубление) - усиление проблемы
- Solution (решение) - предложение продукта

**Правило Before/After/Bridge**
- Before - текущее состояние клиента
- After - желаемое состояние
- Bridge - как продукт поможет достичь цели

### 4. Специфические техники

**Техника "Потому что"**
- Всегда объясняйте причины
- "Скидка 50% потому что..."
- "Бесплатно потому что..."

**Техника "Как будто"**
- "Как будто у вас есть личный консультант"
- "Как будто вы уже достигли цели"
- Создание ощущения обладания

**Техника "Ограниченность"**
- "Только для первых 100 покупателей"
- "До конца недели"
- "Эксклюзивно для подписчиков"

## Практические примеры

### Пример 1: Заголовок с эмоциональным триггером
**Плохо:** "Курс по маркетингу"
**Хорошо:** "Как увеличить продажи в 3 раза за 30 дней (даже если вы новичок в маркетинге)"

### Пример 2: Использование принципа якорения
**Плохо:** "Консультация стоит 5000 рублей"
**Хорошо:** "Обычно час работы маркетолога стоит 10000 рублей, но сегодня консультация всего 5000"

### Пример 3: Техника PAS
**Проблема:** "Устали от низких продаж?"
**Усугубление:** "Каждый день бездействия - это потерянные клиенты и деньги"
**Решение:** "Наш курс покажет, как удвоить продажи за месяц"

### Пример 4: Конкретность в описании
**Плохо:** "Много довольных клиентов"
**Хорошо:** "Более 1000 клиентов увеличили продажи на 150% за первые 3 месяца"
