# Временные аспекты восприятия

## Критичные первые 200-500 мс

### Суть
Пользователь принимает решение в первые полсекунды на подсознательном уровне. Первое впечатление критично для всего дальнейшего взаимодействия.

### Применение
- Первое впечатление критично
- Загрузка должна быть мгновенной
- Главный экран должен сразу показывать ценность
- Цветовая схема и типографика работают мгновенно
- Оптимизировать первые 500 мс взаимодействия

### Практический совет
Оптимизируйте первые 500 мс взаимодействия - это определяет весь дальнейший опыт.

## Цвета обрабатываются за 50-150 мс

### Суть
Пик нейронной активности при цветовосприятии происходит в первые 50-150 мс - в период подсознательного восприятия.

### Применение
- Цвета влияют на решения до осознания
- Первое цветовое впечатление критично
- Цветовая схема должна работать мгновенно
- Подсознательное воздействие цветов сильнее сознательного

### Практический совет
Цвета работают на подсознательном уровне - используйте это.

## Периферийное восприятие работает за 100 мс

### Суть
Периферийные визуальные раздражители активируют мозг уже в первые 100 мс через прямую связь с лимбической системой.

### Применение
- Создавать "эмоциональные аллеи" в интерфейсах
- Размещать позитивные элементы по краям экрана
- Использовать периферийное зрение для подсознательного воздействия
- Эмоциональные элементы на периферии влияют на центральный контент

### Практический совет
Периферийное восприятие работает быстрее сознания - используйте это.

## Решения принимаются за миллисекунды

### Суть
В первые секунды после входа в магазин покупатель принимает решение о направлении движения за миллисекунды и бессознательно.

### Применение
- Первые секунды критичны для направления покупателей
- Решения принимаются бессознательно
- Эмоциональные триггеры должны работать мгновенно
- Создавать четкие эмоциональные указатели направления

### Практический совет
Первые секунды = бессознательные решения о направлении.

## Решения принимаются "по ощущению"

### Суть
В перенасыщенном мире решения часто принимаются "по ощущению", а рациональные объяснения приходят постфактум.

### Применение
- Создавать правильные "ощущения" у покупателей
- Не полагаться на рациональные аргументы
- Работать с интуитивными реакциями
- Понимать, что объяснения приходят после решений

### Практический совет
"Ощущения" важнее рациональных аргументов.

## Гомо экономикус не существует

### Суть
Рациональный покупатель - это миф. Поведение человека регулируется бессознательными нейронными процессами, а не рациональными решениями.

### Применение
- Не рассчитывать на рациональные аргументы
- Работать с бессознательными процессами
- Фокусироваться на эмоциях, а не на логике
- Понимать, что решения принимаются подсознательно

### Практический совет
Рациональность - это иллюзия, работайте с эмоциями.

## Все сигналы проходят через лимбическую систему

### Суть
Все сигналы сначала проходят через лимбическую систему (эмоционально значимо/незначимо), и только затем переходят на уровень сознания.

### Применение
- Каждый элемент должен вызывать эмоциональную реакцию
- Работать с лимбической системой напрямую
- Создавать эмоционально значимые сигналы
- Избегать эмоционально нейтральных элементов

### Практический совет
Лимбическая система = первый фильтр всех сигналов.

## Эмоциональные раздражители декодируются мгновенно

### Суть
Эмоциональные раздражители (фотографии) декодируются в лимбической системе по принципу боль/удовольствие, влияя на восприятие товаров.

### Применение
- Каждый визуальный элемент влияет на эмоциональное состояние
- Создавать "удовольствие" через позитивные образы
- Избегать "боли" через негативные элементы
- Работать с лимбической системой через эмоциональные триггеры

### Практический совет
Лимбическая система = эмоции = решения о покупке.

## Дети - самый сильный эмоциональный триггер

### Суть
Фотографии детей вызывают самую сильную подсознательную реакцию в первые 150 мс.

### Применение
- Использовать изображения детей для максимального эмоционального воздействия
- Размещать детей в периферийной зоне для подсознательного влияния
- Связывать продукты с семейными ценностями через детей
- Дети работают лучше смайликов и других эмоциональных элементов

### Практический совет
Дети - самый мощный эмоциональный триггер для подсознания.
